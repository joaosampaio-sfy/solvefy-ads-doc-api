## Base URLs

```
# Usuários e Subcontas
https://ads.api.dev.solvefy.com

# Campanhas, Relatórios, Billing e Webhooks
https://ads.api.dev.solvefy.com/ads/v1
```

## Autenticação rápida

Toda requisição precisa de dois headers obrigatórios:

| Header | Valor |
|---|---|
| `Authorization` | `Bearer <sua-api-key>` |
| `dp-subaccount-id` | `<token-da-subconta>` |

<Info>
  Acesse [portal.solvefy.com/api-keys](https://portal.solvefy.com/api-keys) para criar sua chave de API.
</Info>
